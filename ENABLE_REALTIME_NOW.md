## 🐛 If It's Not Working

### Check browser console:
1. Open DevTools (F12)
2. Go to Console tab
3. Look for messages like:
   - "Game updated:" ✅ Good!
   - "Subscription error" ❌ Problem

### Common issues:
- **Toggle not green:** Go back and enable it
- **Code not found:** Make sure you're using the exact code (case-sensitive)
- **No updates:** Refresh both browsers and try again
